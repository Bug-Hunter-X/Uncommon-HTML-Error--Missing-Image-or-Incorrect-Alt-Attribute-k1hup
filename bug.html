This is a simple HTML page with a common error:

<!DOCTYPE html> <html> <head> <title>My Webpage</title> </head> <body> <p>Hello, world!</p> <img src="image.jpg" alt="My Image"> </body> </html>

The problem is that the image file might not be in the same folder, which results in the image not showing up properly. Also, sometimes an alt attribute is missing. These are common issues to look for.
